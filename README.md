# 4D Quaternion Julia Set Viewer

4次元の数学的世界に存在する神秘的なフラクタル「4次元四元数ジュリア集合（4D Quaternion Julia Set）」を、レイマーチング（Raymarching）を用いてブラウザ上でリアルタイムに3D描画し、自由に探索・操作できるインタラクティブなアートビューアです。高度な数式とWebGL2の描画技術を組み合わせることで、本来は目に見えない4次元空間の美しい断面を、まるで目の前にある彫刻のようにリアルタイムで鑑賞できます。

PCやスマートフォンのブラウザだけでなく、VRゴーグル（Meta Questなど）を使用したWebXRモード（VR/AR）にも対応しています。仮想空間にフラクタルを浮かび上がらせ、手やコントローラーで回転・拡大縮小させたり、ARパススルー機能によって自分の部屋に4次元フラクタルのオブジェを配置するような、未来的な体験が可能です。

複雑な設定なしで楽しめる多様なプリセット形状や自動アニメーション機能、自分だけの美しい瞬間を記録するための高解像度画像書き出し機能（背景透過対応）、そして作成した形状をワンクリックで友達に共有できるURL生成機能など、フラクタルの美しさを最大限に楽しむための機能が豊富に備わっています。

## ✨ 主な機能 (Features)

* **4次元フラクタルのリアルタイム探索**: 4次元の四元数ジュリア集合の形状を、WebGL2を用いたリアルタイム・レイマーチング技術により、驚くほど滑らかな高画質で探索できます。
* **初めてでも迷わないインタラクティブツアー**: 初回起動時に自動で開始される「使い方ガイドツアー」を搭載。各機能の役割や見どころをステップバイステップで案内します（設定UI最下部の「使い方ガイド」からいつでも再起動可能です）。
* **ワンタップで切り替わる美しいプリセット**: 「Deep Blue（深い青の洞窟）」「Jade Artifact（翡翠の遺物）」「Crimson Ring（深紅の輪）」「Golden Amber（黄金の琥珀）」に加え、天使のように神秘的な対称美を持つ「Angel Wings（天使の羽）」など、すぐに素晴らしい体験ができるプリセットを搭載しています。
* **自動アニメーション**: パラメータが滑らかに変化し、生き物のようにうごめくアニメーション機能を搭載。動きのテンポや振幅もプリセットや詳細設定から簡単に切り替えられます。形状が破綻したり消滅したりしない安全制御ロジックも備わっています。
* **モバイル・PCそれぞれに最適化したデザイン**: 画面幅に合わせて自動でレイアウトが最適化されます。特にモバイル環境では、ボトムシート型のスワイプ式メニュー、片手での操作に適した「1列（縦積み）スライダー」や「3ブロック階層メニューレイアウト」などを採用し、高い視認性と操作性を実現しています。
* **WebXRによる没入VRとパススルーAR**: VRゴーグルを使って立体的にフラクタルを鑑賞できるだけでなく、AR機能を使って現実空間の部屋の中にフラクタルを配置することも可能です。
* **高品質・超解像画像の書き出し**: 画面上の美しい瞬間を壁紙サイズなどの超高解像度（最大4096px）で保存できます。GPUの負荷を抑えつつ安全に処理する「タイル分割レンダリング」を実装しており、背景を透過させたPNG画像の出力も可能です。
* **ランダム生成とUndo/Redo**: ボタン一つで予期せぬ形状やカラーパレットをランダム生成。失敗しても最大30件まで操作を「元に戻す（Undo）」「やり直す（Redo）」ことができます。
* **パラメータのURL共有**: お気に入りのフラクタルができたら、その設定をURLパラメータに保存してコピー可能。URLを開くだけで、他のブラウザでも全く同じ形状を復元して鑑賞できます。
* **快適さを維持する自動画質調整 (Adaptive Quality)**: デバイスの処理速度（FPS）を検知し、動作が重くなると自動的に描画解像度を下げ、安定すると自動で元の画質に戻すインテリジェントな画質調整機能を備えています。

## 🎮 操作方法 (Controls)

### PC (デスクトップ)
* **回転**: 画面を**左ドラッグ**
* **ズーム (拡大・縮小)**: マウスの**スクロールホイール** または **中ドラッグ**
* **平行移動**: 画面を**右ドラッグ**
* **設定の表示/非表示**: 画面右下の「設定」ボタンをクリックしてパネルを開閉します。

### モバイル (スマートフォン/タブレット)
* **回転**: 画面を**1本指でスワイプ**
* **ズーム ＆ 平行移動**: 画面を**2本指でピンチ（広げる/狭める）またはスワイプ**
* **設定の表示/非表示**: 画面右下の「設定」ボタンをタップしてパネルを開閉します。

### XR (VR / ARゴーグル)
* **物体の移動と回転（直接グラブ）**: コントローラーの**トリガー（セレクト）を引いたまま**手を動かすことで、フラクタルオブジェクトを直接掴んで位置を動かしたり、手首を回して回転させたりできます。
* **拡大・縮小（両手ピンチ）**: **両手のトリガーを同時に引きながら**、コントローラーの距離を離す（拡大）または近づける（縮小）ことで、直感的にサイズを変更できます。
* **コントローラーのスティック操作**:
  - **右スティック（左右/上下）**: 通常の3D空間での回転（ピッチ・ヨー）を行います。
  - **左スティック（左右）**: 4次元の平面回転（ZW平面）を行い、物体の内部と外部を反転させます。
  - **左スティック（上下）**: オブジェクトのVR表示スケールを拡大・縮小します。
* **ハンドトラッキング (素手操作)**: コントローラーの代わりに、自身の両手を用いてピンチ操作（親指と人差し指をくっつける）を行うことで、掴み操作や両手での拡大縮小が可能です。

## 🎛️ 主要パラメータ (Parameters)

下部の設定パネルから、以下のパラメータを自由に組み合わせて自分だけのフラクタルを作成できます。

### 1. 形状パラメータ (Fractal)
* **C.x / C.y / C.z / C.w (4D複素定数)**: フラクタルの骨格や肉付き、ねじれなど、4次元空間での基本的な「かたち」を決定する最も重要なパラメータです。
* **X軸/Y軸/Z軸回転 (3D回転)**: 通常の3次元空間におけるカメラ周りでのフラクタルの回転（Pitch, Yaw, Roll）です。
* **XW/YW/ZW平面回転 (4D回転)**: 4次元空間独自の回転。第4の軸（W軸）を用いて空間を回転させます。物体が内側から裏返るように見えたり、空間がねじれるような異次元の形状変化を楽しめます。
* **視野角 (FoV)**: カメラの視野の広さを調節します。値を大きくすると遠近感が強調され、小さくすると望遠レンズのようになります。

### 2. 表示・質感パラメータ (Material & Style)
* **ベースカラー**: フラクタルの基準となる色をカラーピッカーで指定します（色相・彩度へ自動反映）。
* **明るさ**: 全体的な光の反射と輝度を調節します。
* **影の濃さ (AO)**: くぼんでいる部分に影（アンビエントオクルージョン）を落とし、複雑な細部の立体感を際立たせます。
* **光沢感 (Specular)**: 表面のツヤを調整し、マットな質感から金属のような鋭い輝きまで表現できます。
* **背景色 / 背景不透明度**: フラクタルを浮かび上がらせる宇宙のような背景の色と透過度を設定します。
* **ズーム**: カメラのズーム倍率を変更します。

### 3. 自動アニメーション設定 (Animation)
* **全体の速度**: アニメーション全体の変化スピードを調整します。
* **変形の大きさ (振幅)**: 形状がアニメーションで変形する最大の揺れ幅を設定します。
* **個別軸の速度比 / 振幅比 / 初期位相**: C.x〜C.wの各成分が変化する個別のテンポやズレを細かく制御できます。

## 👓 XR機能 (WebXR Features)

本アプリは WebXR 規格に対応しており、Meta Quest などの互換デバイスからブラウザ（Meta Quest Browserなど）を介して直接起動できます。

* **VRモード (没入型体験)**: 没入感のある暗黒の仮想宇宙空間に、巨大なフラクタルオブジェクトを配置して鑑賞できます。
* **ARモード (パススルー体験)**: 現実の部屋の中にフラクタルオブジェクトを重ね合わせて配置できます。ARモード移行時は、フラクタルの背景が自動的に完全透過されます。
* **ハンドトラッキング**: コントローラーを持たず、素手でのピンチ（掴む）動作だけで直感的な操作が可能です。
* **6-DoFコントローラー操作**: 左右のコントローラーを使い、空間内でフラクタルオブジェクトを直接「掴んで移動・回転」「両手で伸縮して拡大縮小」する直感的な体験をサポートしています。

## 🛠️ 技術スタック (Tech Stack)

* **Three.js**: 3D描画エンジンのコア。WebXRセッションの管理やカメラコントロールを支えています。
* **WebXR API**: デバイスのVR/ARモード、6-DoFコントローラー、およびハンドトラッキングをブラウザ上で実行します。
* **GLSL (OpenGL Shading Language)**: 4次元フラクタルの複雑な数式とレイマーチング処理を、GPUで高速並列計算するためのフラグメントシェーダーコードです。
* **Vite**: 高速な開発環境と、成果物を最適化して配信するためのビルドツール。

## 🚀 開発とローカル起動 (Development)

ローカル環境で本プロジェクトを起動・検証・ビルドする方法です。

### 動作要件
* **Node.js** (v18以上推奨)

### 1. インストール
プロジェクトのルートディレクトリで以下を実行して依存パッケージをインストールします。
```bash
npm install
```

### 2. 開発用ローカルサーバーの起動
開発用サーバーを起動します。
```bash
npm run dev
```
起動するとターミナルにローカルURL（例: `http://localhost:5173/`）が表示されます。ブラウザでそのアドレスを開いてください。

### 3. プロダクションビルドの作成
公開用の最適化された静的ファイルを生成します。
```bash
npm run build
```
ビルドが完了すると、`dist/index.html` にすべてのHTML、CSS、JavaScriptを単一のファイルにパッケージングした成果物が出力されます（`vite-plugin-singlefile` を使用）。このファイルをWebサーバーに配置するだけで公開可能です。
